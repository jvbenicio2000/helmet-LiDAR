# Modelos 3D

Esta pasta contém os arquivos de modelo 3D para a impressão das peças do projeto Helmet-LiDAR.

## Suporte do Sensor Ultrassônico

- **Arquivo:** `bloco_3_furos_v3.stl`
- **Descrição:** Este componente é o suporte projetado para acoplar o sensor ultrassônico (HC-SR04) ao motor de passo, permitindo a sua rotação para a varredura do ambiente.

## Outros Componentes

- **Arquivo:** `nema-17-mount-v1.stl`
- **Descrição:** Suporte para o motor de passo NEMA 17, responsável pela rotação do conjunto do sensor.

- **Arquivo:** `anel_complexo_v4.stl`
- **Descrição:** Anel superior do conjunto giratório, possivelmente para fixação de componentes.

- **Arquivo:** `cilindro_furos_tangentes_v2.stl`
- **Descrição:** Cilindro com furos tangenciais, provavelmente para passagem de fios ou como parte do acoplamento mecânico.

- **Arquivo:** `anel_peca_inferior.stl`
- **Descrição:** Anel inferior do conjunto giratório, complementando o `anel_complexo_v4.stl`.

- **Arquivo:** `bloco_furo_centralizado.stl`
- **Descrição:** Bloco com furo central, que pode ser usado como espaçador ou para acoplamento no eixo do motor.
