# Lista de Materiais (Bill of Materials)

Esta é a lista de componentes e materiais necessários para a montagem completa do projeto Helmet-LiDAR.

| Categoria | Componente | Quantidade | Observações |
|---|---|---|---|
| **Microcontroladores** | Raspberry Pi Pico W | 2 | Um para o sistema principal e outro para funções auxiliares ou comunicação. |
| **Sensores** | Sensor Ultrassônico HC-SR04 | 1 | Para medição de distância. |
| | Módulo GPS NEO-6M | 1 | Para funcionalidades de geolocalização. |
| **Atuadores** | Motor de Passo NEMA 17 | 1 | Para a rotação do conjunto do sensor. |
| | Driver para Motor de Passo TMC2209 | 1 | Controla o motor de passo com precisão (microstepping). |
| | Motores de Vibração de Alta Intensidade | 4 | 12V, com massa desbalanceada para feedback tátil. |
| | Driver Ponte H L298N | 2 | Para controlar os quatro motores de vibração. |
| **Estrutura** | Capacete de Segurança | 1 | Base para a montagem de todos os componentes. |
| | Filamento para Impressão 3D (PLA/PETG) | - | Para imprimir todas as peças e suportes customizados. |
| **Eletrônica** | Placa de Circuito Impresso (PCB) | 1 | Para uma montagem final e robusta dos componentes eletrônicos. |
| | Fiação e Conectores Diversos | - | Para realizar todas as conexões elétricas. |
| **Alimentação** | Bateria / Fonte de Alimentação | 1 | Ex: Power Bank de alta capacidade ou conjunto de baterias Li-ion. |
