# Lista de Materiais (Bill of Materials)

Esta é uma lista preliminar dos componentes necessários para o projeto Helmet-LiDAR.

| Componente | Quantidade | Observações |
|---|---|---|
| Sensor Ultrassônico (HC-SR04) | 1 | Para medição de distância. |
| Motor de Passo (28BYJ-48 com driver ULN2003) | 1 | Para rotacionar o sensor ultrassônico. |
| Motores de Vibração (Tipo moeda ou similar) | 4 | Para feedback tátil. |
| Microcontrolador (Arduino Nano, Uno ou ESP32) | 1 | Cérebro do projeto. |
| Módulo de Ponte H (L298N ou similar) | 1 | Para controlar os motores de vibração (se necessário). |
| Capacete de segurança | 1 | Base para a montagem dos componentes. |
| Fios Jumper | - | Para as conexões. |
| Fonte de alimentação (Bateria ou Power Bank) | 1 | Para alimentar o circuito. |
